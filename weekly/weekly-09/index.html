工具#

1、Whisper

一款由 Openai 发布的 听译软件，据说最大的模型对于英语的识别准确度能达到 99%.

实际测试下来发现，虽然随着模型的增加准确度不断提升，但是还存在两个问题

  • 断句的判断还存在缺漏
  • 某些背景音会被误识别进去

2、 Share X

一款强大截图软件，可以自定义非常多的截图处理工序，并实现自动化处理。

使用这款截图软件，起因是由于我在周报第四期推荐的 TinySnap 浏览器插件居然进一步阉割了免费版功能。

起初只有云同步和修改截图、批注需要开通会员，估计是开发者一看用户都选择白嫖，索性把背景渐变配色也给纳入了 Pro。

何况这款浏览器插件还只支持买断制 49 美元。遂弃、已经无法用吃相难看来形容了。

现在使用的 Share X，通过自定义截图动作，也可以实现类似的效果，关键是完全免费。

截图效果展示

这款软件还支持录屏、滚动截屏等功能，有兴趣的朋友可以自行了解。

3、Semantra

一款支持多文档、多模型的 PDF 自然语义搜索项目

安装步骤也非常简单，使用作者的文档介绍给出 Pipx 包安装即可。

这款工具给出了各种模型，具体可以参考 这里

实测下来，中文依旧支持较差。而模型当中，使用 Openai Api Key 效果最好，但这种方式需要花费一定的 Api 额度。

我测试了一本两百页的英文文献，使用 GPT 3.5 的模型，向量化一次性花费了 0.1 美元，个人认为还算可以接受的。
