何为Markdown#

Markdown是一种轻量标记语言，简单易上手，去繁就简，采用纯文本的形式方便跨平台输出。

Markdown 语法#

可以查看这里简单一图了解Markdown语法：传送门

标题#

格式：#xn+空格+文本内容，其中n表示级数，一般支持到六级

# 一级标题
## 二级标题

斜体#

格式：*+文本内容+*

*这是斜体*

粗体#

格式：**+文本内容+**

**这是粗体**

分割线#

格式: 由三个* * *或者- - -组成

---

示例#

删除线#

格式：~~+文本内容+~~

~~这是带删除线的文本~~

列表#

有序#

格式：数字+.+空格+文本内容

1. 第一点
2. 第二点
示例#
  1. 第一点
  2. 第二点

无序#

格式：-+.+空格+文本内容

- 无序列表一
- 无序列表二
示例#
  • 无序列表一
  • 无序列表二

缩进#

上述所说两种列表，如果需要子级列表，可以回车后使用Tab来生成二级列表

1. 第一点
	1. 第一点的第一点
	2. 第一点的第二点
2. 第二点
	- 第二点的某一点
	- 第二点的某一点
示例#
  1. 第一点
    1. 第一点的第一点
    2. 第一点的第二点
  2. 第二点
    • 第二点的某一点
    • 第二点的某一点

HTML 语法#

由于Markdown支持Html语言，所以有一些用法，是可以直接应用过来的，这里简单记录几个常用的。

Markdown本身注重的就是减少排版文本所造成输出干扰，所以太频繁使用Html语言多少有点本末导致的感觉…

注释#

格式：<!- -+注释的内容+- - >

示例#

< ! – 这是一段注释文字 –>

换行#

格式：直接插入标签<br>即可。

居中#

格式：<center>+需要居中的对象+</center>.

彩色文本#

格式：<font color=orange>这是一段橘色文本</font>，这里color对应值可以是 英文单词，16进制，rgb，rgba.

插入音频#

<audio controls="controls" preload="none" src="音频链接"></audio>

插入页面#

<iframe width=600 height=400 src="插入的链接" scrolling="auto" border="0" frameborder="no" framespacing="0" allowfullscreen="true"> </iframe>

应用#

有时候博客网站想要插入一首歌曲，网易云便提供这种外链调用的方式，具体使用模板如下：

<center><iframe frameborder="no" border="1" marginwidth="0" marginheight="0" width=90% height=86 src="//music.163.com/outchain/player?type=2&id=1350658589&auto=0&height=66"></iframe></center>

打开网页版网易云，选择生成外链播放器，修改本代码中的id后对应值即可。其中的<center>标签意为元素居中，而width=90%属性意为自适应页面大小，宽度为内容区域的90%.

示例#

<center><iframe frameborder="no" border="1" marginwidth="0" marginheight="0" width=90% height=86 src="//music.163.com/outchain/player?type=2&id=29769433&auto=0&height=66"></iframe></center>

软件推荐#

Typora#

一款界面优美、实时渲染的Markdown软件。

Obsidian#

Obsidian_是基于Markdown文件的本地知识管理软件，并且开发者承诺Obsidian对于个人使用者永久免费，目前支持Win、Mac、ios、安卓等平台。

Mweb#

专业的 Markdown 写作、记笔记、静态博客生成软件，支持一键发布到少数派、CSDN、Wordpess等博客平台。
